# results/

- `flags/` — one report per audited paper: every (claim, citation, abstract)
  triple the LLM could not find quoted support for. This is a review queue,
  not a verdict — every flag here needs manual adjudication before it counts
  toward any reported rate.
- `examples/` — full evidence chains (sentence → cited paper → abstract text)
  for the 2-3 sharpest confirmed cases, kept unanonymized internally.
  Anonymize only when copying into the manuscript draft, not here.
